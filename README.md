# Archnemix – YouTube Shorts Generator

Frontend UI for the Archnemix Shorts ecosystem.

## Features
- Script → Short video pipeline
- Multiple background categories
- Client-side rate limiting
- Clean, fast UI (Vite + React + Tailwind)

## Tech Stack
- React + TypeScript
- Vite
- Tailwind CSS
- Cloudflare Pages

## Note
This repository contains **frontend code only**.
Backend APIs are protected and not exposed here.
